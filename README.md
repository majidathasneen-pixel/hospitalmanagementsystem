# Hospital Management System

This is a Hospital Management System web application developed using Django. It helps to manage hospital activities like patients, doctors, appointments, prescriptions, and medical records in a digital way.

## Features

- Admin, Doctor, and Patient login system
- Patient registration and management
- Doctor management
- Appointment booking system
- Prescription management
- Medical records management
- Dashboard for users

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite Database
